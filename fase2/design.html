Etiqueta Semánticas

ayudan a definir la estructura del documento y permiten que las páginas web sean mejor indexadas por los buscadores. Una etiqueta se califica como semántica si tiene que ver el significado, es decir, si nos informa sobre lo que trata su contenido

Bernal, D., Castro, A., & González, J. (2014). Web semántica, más de una década de su aparición.

CSS Grid

Es una técnica de las Hojas de Estilo en Cascada que permite a los desarrolladores web crear diseños complejos y adaptables con mayor facilidad en todos los navegadores

Ajmi, A., Kumar, N. y Roworth, A. (2017). Diseñar con CSS Grid Layout . SitePoint.
