Etiquetas semánticas

ayudan a definir la estructura del documento y permiten que las páginas web sean mejor indexadas por los buscadores. Una etiqueta se califica como semántica si tiene que ver el significado, es decir, si nos informa sobre lo que trata su contenido.

https://www.aulaclic.es/articulos/html5-semantica_1.html

CSS Grid

es una técnica de las Hojas de Estilo en Cascada que permite a los desarrolladores web crear diseños complejos y adaptables con mayor facilidad en todos los navegadores y se puede utilizar para lograr muchos diseños diferentes.​

https://developer.mozilla.org/es/docs/Web/CSS/CSS_Grid_Layout
